# kyawminthu20.github.io

This repository is the working space for a personal GitHub Pages website.

The goal is modest: build a small, honest site that introduces Kyaw Min Thu, shares selected experience and projects, and stays clear of inflated claims.

## Current State

- `planning/index.html` is the current website prototype. It is a single HTML file with inline CSS and JavaScript.
- `RAG/kmt_career.yaml` is the main structured source for career history, skills, education, and contact details.
- `software_projects/CS453Summer2024MobileProgramming/` contains Android coursework and supporting media that may be reused as portfolio material.
- `project_state/` tracks the current phase, change history, environment notes, and working instructions for this repo.

This is not a finished portfolio yet. It is still a planning and content-organizing repository.

## Tone And Content Direction

The site should be:

- humble
- factual
- easy to read
- specific about actual work done

The site should avoid:

- exaggerated claims
- aggressive self-promotion
- skill bars or confidence scores presented as fact
- crowded pages that try to show everything at once

## Near-Term Plan
